Vetores ou arrays

O arrays são uma matriz de variáveis (como se fosse um tipo de lista), em que cada variável tem um índice.

Podemos imaginar um array como uma caixa que possui várias outras caixas dentro e cada uma delas possui um valor.

Array

O array é declarado entre colchetes "[]", e podem guardar qualquer valor dentro de seus índices (inclusive outros arrays).

Exemplo: array = ['string', 1, true, false, ['array1'], ['array2']...]

Observação:
Cada índice é separado por vírgula;
O índice só é acessado por um número inteiro, onde 0 é o primeiro índice.

Manipulando Arrays

O array possui uma série de métodos para manipulá-lo.

forEach()

Executa uma função para cada índice de um array. Faz uma repetição para cada item ou índice dentro de um array.

Exemplo: array.forEach(function(item, index){console.log(item, index)});

push()

Adiciona item no final do array; insere um novo dado dentro do array

Exemplo: array.push('novo item');
console.log(array);

pop()

Remove item no final do array

Exemplo: array.pop();
console.log(array);

shift()

Remove item no início do array

Exemplo: array.shift();
console.log(array).

unshift()

Adiciona item no início do array

Exemplo: array.unshift('novo item no início');
console.log(array);

indexOf()

Retorna o índice de um item dentro de um array.

Exemplo: console.log(array.indexOf(true));

splice()

Remove ou substitui um item pelo índice

Exemplo: array.splice(0,3);
console.log(array);

slice()

Retorna uma parte de um array existente

Exemplo: let novoArray = array.slice(0,3);
console.log(novoArray);

Objetos

Dados que possuem propriedades e valores que definem suas características.

Deve ser declarado entre chaves "{}"

Exemplo: Imagine uma xícara azul. Ela tem cor, pode ter vários tamanhos e funções. Pode ser declarada da seguinte forma:
var xicara = {
cor: 'azul',
tamanho: 'p',
funcao: tomarCafe()
}

Manipulando objetos

Propriedades de objetos podem ser atribuídas à variáveis, facilitando a manipulação do objeto. Chamamos isso de desestruturação.

Exemplo
var xicara = {cor: 'azul', tamanho: 'p', funcao: tomarCafe()}

var cor = xicara.cor;
var tamanho = xicara.tamanho;
var funcao = tomarCafe();

Exemplo 2

let object = { string: 'string', number: 1, boolean: true, array: 'array', objectInterno: { objectInterno: 'objeto interno'}};
console.log(object);

Outra forma de fazer a desestruturação é utilizando chaves ao declarar a variável

Exemplo: var {cor, tamanho, funcao} = xicara;
