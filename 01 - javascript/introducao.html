Introdução ao JavaScript

  • Linguagem de script e multiplataforma
  • Baseada em Protótipos
  • Multiparadigma
  • Client Side
  • Capacidade de interação com elementos de uma página HTML
  • Usado no desenvolvimento de páginas e aplicativos mobile híbrido
  • Segue o padrão ECMAScript

Evolução do JavaScript

O ECMAScript 6 trouxe muitas atualizações para o JavaScript. Por exemplo, essa atualização permite a utilização do Paradigma da Orientação a Objetos.

Aplicações

Apesar do JavaScript ser considerada a Linguagem da Web, também é possível desenvolver:

  • Aplicativos Mobile
  • Smartwatches
  • Games
  • Internet of Things
  • APIs
