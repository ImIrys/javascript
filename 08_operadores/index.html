Tipos de operadores

Operadores, no JavaScript, são sinais que utilizamos para manipular um dado valor ou para validar uma certa condição.

    Os principais operadores utilizados são:
  • Atribuição
  • Aritmética
  • Comparação
  • Lógica
  • Condicional

    • Atribuição

    É muito importante saber como atribuir valores a uma variável (endereço de memória).

    1. Operador =

      2. Operador =
      Exemplo
      x = y
      Equivalente a
      x = y

    2. Operador +=

      3. Operador +=
      Exemplo
      x += y
      Equivalente a
      x = x + y (adiciona o valor da variável a outra variável)

    3. Operador *=

      4. Operador *=
      Exemplo
      x *= y
      Equivalente a
      x = x * y (multiplica o valor da variável a outra variável)

    4. Operador /=

      5. Operador /=
      Exemplo
      x /= y
      Equivalente a
      x = x / y (divide o valor da variável a outra variável)

    5. Operador %=

      6. Operador %=
      Exemplo
      x %= y
      Equivalente a
      x = x % y (retorna o resto da divisão entre as variáveis)

    Aritmética

    Operadores comuns que existem na Matemática.

    1. Operador de Adição " + "
    2. Operador de Subtração " - "
    3. Operador de Multiplicação " * "
    4. Operador de Exponencial " ** "
    5. Operador de Divisão " / "
    6. Operador de Módulo " % "
    7. Operador de Incrementar " ++ "
      8. (Adiciona 1 à variável)
    8. Operador de Decrementar " -- "
      9. (Subtrai 1 da variável)

    Comparação

    1. Operador Igual a " == "
      2. (Pergunta se os valores são iguais mas não faz verificação de Tipo)
    2. Operador Mesmo valor e Mesmo Tipo " === "
      3. (Pergunta se os valores são iguais e faz a verificação de Tipo. Geralmente usado no Mercado de Trabalho)
    3. Operador Diferente " != "
    4. Operador Valor e Tipos Diferentes " !== "
    5. Operador Maior que " > "
    6. Operador Menor que " < "
    7. Operador Maior ou Igual " >== "
    8. Operador Menor ou Igual " <== "

    Lógica

    1. Operador "e" lógico " && "
      2. (Temos uma afirmação e outra afirmação e deve ser usado o "&&" se elas são verdadeiras)
    2. Operador "ou" lógico " || "
      3. (Se uma das afirmações comparadas for verdadeira usar o "||")
    3. Operador "não" lógico " ! "
      4. (Usado se quisermos acessar o dado o oposto do dado que está sendo passado após o "não" lógico)

    Condicional (if ternário)

    Chamado de if Ternário

    É o mesmo que escrever um if else

    (condition) ? expression1 : expression2

    Lê-se: Essa condição é verdadeira? Se sim, executar a 1º expressão, se não, executar a 2ª expressão.
