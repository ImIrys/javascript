1. Estrutura de projeto simples

Veremos uma estrutura de projeto básica para a criação de projetos que utilizam JavaScript, HTML e CSS.

  • Index
O index, que significa índice em inglês, é o nome padrão utilizado para o local onde fica a entrada principal do código. É o lugar onde teremos acesso às diferente páginas (atalhos para as outras páginas).

  • Arquitetura
Comumente, quando se tem projetos que utilizam diversos tipos de arquivos é preciso separá-los por pastas. Isso é uma noção básica de Arquitetura.

A arquitetura é como está desenhada toda a estrutura dos arquivos do projeto.

  • Assets
Assets são, basicamente, todos os componentes da página HTML (arquivos em css, javascript e etc) e que não são arquivos HTML.

O arquivo HTML é o principal, então deve ficar na raíz do projeto, separado dos outros arquivos.

OBS: Essas nomenclaturas básicas são dadas para projetos pequenos e iniciantes.

2. Inserção do Javascript no HTML

  • Inserindo arquivo CSS

Dentro da tag head, antes da tag title, inserir a tag link.Essa tag contêm 2 atributos: rel e href.

O rel significa o relacionamento que existe entre o arquivo que será chamado (style.css) e o arquivo no qual estarei o chamando (index.html). O relacionamento dela com ele é que ela é uma stylesheet

O href é a referência, onde está localizado esse arquivo.

Exemplo: link rel="stylesheet" href="style.css" />

OBS: Em alguns arquivos javascript ou bibliotecas que forem baixadas pode ser aconselhado colocar esses arquivos antes do encerramento da tag html. Porque quando o documento é renderizado na web ele será lido proceduralmente de cima para baixo. E se houver um script muito grande, ele começará a rodar e provavelmente buscará interagir com elementos da tela que não foram renderizados.

  • Inserindo arquivo Javascript
A tag utilizada para arquivos javascript é a script. O atributo que indica origem é o src
