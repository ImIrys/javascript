Estrutura de Dados

O JavaScript é uma linguagem de tipagem dinâmica. Ou seja, antes de declarar um valor, não é necessário especificar o tipo dele.

Observação: Sempre que quiser verificar o tipo de algum dado é só digitar: typeof variavel

Tipos

Existem 2 tipos de dados em JavaScript:

  1. Dados Primitivos
  2. Dados Compostos

Dados Primitivos

  1. Não possuem método dentro deles
  2. Escritos com letras minúsculas

Podem ser:

  • String
  • Números
  • Valor Booleano (true/false)
  • Null
  • Undefined

Dados Não Primitivos

  • Objetos
  • Arrays

Strings

  1. Utilizada para textos
  2. Valores são declarados entre aspas " " ou crases ` `
  3. Pode ser usada como objeto
  4. É um tipo de dado iterável

Exemplo:

                    
                        let nome = "Valentina";
                        let sobrenome = "Sobral";

Concatenação de strings

                        
                            nome.concat(sobrenome);
                            let concatenado = nome.concat(sobrenome);

                            console.log(concatenado); // vai aparecer "ValentinaSobral"

Outras formas de concatenação

Exemplo: aparecer "Valentina Sobral"

1ª forma:

                        
                            concatenado = nome + " " + sobrenome;
                            
                            console.log(concatenado);

2ª forma:

                        
                            concatenado = `${nome} ${sobrenome}`;

                            console.log(concatenado);

Exemplo: aparecer
"Valentina"
"Sobral"

1ª forma:

                        
                            concatenado = nome + "\n" + sobrenome;

                            console.log(concatenado);

2ª forma:

                        
                            concatenado = `${nome} 
                            ${sobrenome}`;

                            console.log(concatenado);

String como objeto

Exemplo 1:

                        
                            let exemplo = "Hello" // declaração de um tipo primitivo

Exemplo 2:

                        
                            let exemplo = new String ("Hello") // String com letra maiúscula é um objeto.

No exemplo 2, a variável exemplo retornará um objeto do tipo String, com tudo o que está dentro dos índices. Também virá algumas propriedades dentro do prototype (possui todos os métodos que podemos utilizar numa String.)

Observação: Não tem problema se quisermos utilizar os métodos dos objetos String dentro do tipo primitivo string. Porém, é preciso entender que existe a diferença que o objeto String é muito maior do que o tipo primitivo string (mais rápido para compilar e ler).

String como dado iterável

É possível verificar o que existe em cada índice, em cada caractere da String.

Exemplo: Saber o que está no índice 1

                        
                            nome = "Irys";
                            nome[1] // será a letra "r"

Exemplo: Porém, quando perguntamos o tamanho da variável:

                        
                            nome.length // aparecerá o valor "4"

Observação: Não confundir tamanho (começa sempre do 1) com índice (começa sempre do 0).

Printar aspas ""

Imprimir o valor de ""

Exemplo:

                        
                            concatenado = "\"";
                            console.log(concatenado);

Outros métodos

Método split

O método split vai separar os argumentos da string e vai retornar os valores dessa frase, não modifica a variável.

Exemplo: ao passar uma string vazia, o split vai separar tudo (cada espaço, cada vírgula).

                                
                                    let frase = "Olá, mundo.";
                                    frase.split("");

Observação: ao passar uma string com espaço, o split vai separar somente quando houver espaço.

Método includes

Utilizando o método includes é possível saber se a variável possui algo em específico.

Exemplo:

frase.includes("mundo");

Método startsWith

Com esse método, podemos descobrir se a variável começa com algo em específico

frase.startsWith("O") //vai retornar o valor booleano true

Método endsWith

Com esse método, podemos descobrir se a variável termina com algo em específico

frase.endsWith("m") //vai retornar o valor booleano false

Método replace

É possível fazer modificações na variável String utilizando esse método.

Exemplo:

                            
                                frase.replace(",", "!"); // vai retornar uma string modificada: Olá! mundo.

Ele retorna um valor modificado. E se quisermos utilizar esse valor modificado é preciso atribuí-lo a um endereço de memória

Exemplo:

                            
                                let stringModificada = frase.replace(",", "!");

Resumo
