Console

É possível utilizar o console de 2 formas:

  • Numa página da Web
  • No Terminal

Console numa página da Web

Em seu navegador (utilizei o Chrome) ao clicar com o botão direto do mouse e clicar em "Inspecionar", abrirá uma página cheia de ferramentas. Em uma das abas, encontra-se o Console.

Ao clicar no Console, aparecerá uma tela em branco. Nessa tela, é possível escrever coisas. Por exemplo, é possível declarar variáveis e fazer operações com ela. Também é possível declarar e testar funções e outras operações.

Geralmente, utilizamos o console numa página da Web quando estamos criando uma aplicação Web e/ou lidando com andamentos Client Side (elementos que serão apresentados para o cliente ou usuário, como interface ou coisas interativas).

Console no Terminal

Outra forma de utilizar o console é utilizando o Node Js.

O Node Js é uma ferramenta que ajudará a rodar o JavaScript do lado do servidor (da pessoa que está desenvolvendo).

Em qualquer terminal será possível utilizar o console e visualizar o output do código JavaScript.

No VSCode, no arquivo javascript, na linha 1 chame o console da seguinte forma:

console.log('dizer o que quero ver');

Exemplo:
console.log('Hello World!')

Abra um novo terminal. Para visualizar o output do código javascript será necessário entrar no local onde esse arquivo está salvo.

No terminal, para visualizar onde você está digite ls.

Digitar cd + nome da pasta/ e apertar Enter.

Fazer isso (cd+nome da pasta/) até encontrar a pasta que está o arquivo js.

Após encontrar o arquivo js, digitar node + nome do arquivo.js para rodar o arquivo js. E ele vai mostrar a mensagem contida no console.log.

Basicamente, é dessa forma que será possível ver o output do console no terminal.

OBS: Dessa forma não é possível ver em tempo real como no console na página web.

Exemplo:

function returnEvenValues(array) {
let evenNums = [];
for(let i = 0; i < array.length; i++) {
if(array[i] % 2 === 0) {
evenNums.push(array[i]);
} else {
console.log(`${array[i]} não é par!`);
}
}
console.log('Os números pares são:', evenNums);
}
let array = [1, 2, 4, 5, 7, 8];

returnEvenValues(array);
